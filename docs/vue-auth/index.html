Usage

Registration

Easily handle user registrations, email verifications, and more with built-in support for validation and error handling.

Login

Seamlessly integrate login functionality with Vue's Composition API and full TypeScript support.

Logout

Effortlessly log users out of your app and manage their sessions securely across multiple pages.

Password Recovery

Provide users with a simple way to reset their passwords with secure token-based recovery flows.

Automatic Redirection

Automatically redirect users based on their authentication state to ensure smooth navigation within the app.

User State Management

Keep track of user authentication states and roles with reactive and global state management.
